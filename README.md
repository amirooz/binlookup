## Installation and use

```
$ git clone https://github.com/amirphp7/binlookup.git
```
```
$ cd binlookup/app
```
```
$ php app.php input.txt
```
```
$ Unit Test
```
```
$ composer install
```
**If missing any packages**
```
$ composer update
```
